VS Code快速配置C++环境

发表于2022-10-29|更新于2022-10-29
|字数总计:390|阅读时长:1分钟|阅读量:

前言

因为 Visual Studio 实在是太大了，想找个替代品，但是 Dev-C++没有代码自动补全。在网上一番搜索后发现 VS Code 也可以作为 C++ IDE。就立马去折腾。但是给 VS Code 配置环境是真的麻烦，折腾了一天。今天在 B 站刷视频的时候无意看到一个评论，直接让我解决了配置问题。于是就写下这篇文章记录。

准备工具：

​ 1.Visual Studio Code

​ 2.MinGW

​ 3.快速配置工具—— VS Code Config Helper 工具由国人制作，作者是谷雨同学。

配置步骤：

1.下载 Visual Studio

这里不多做赘述，需要本教程的都应该已经安装好 VS Code 了吧？

2.点击下载 MinGW。

因为是外网，下载速度可能会比较慢，我就放出蓝奏云链接，版本为 GCC-8.1.0。

注意：如果是打开上面的”下载 MinGW”的话不要点 Download Lastest Version! 网页拉下去选择下图中圈出来的下载

下载后解压就好了。

3.下载快速配置工具 VS Code Config Helper

下载地址

（VS Code Config Helper 支持 Windows，MacOS 和 Linux）

下载打开工具后就按照指示配置就好了。注意！工具文件夹目录不要有中文，不然无法编写 cpp!

配置好后打开 VS Code，工具会自动生成测试 cpp 文件.编译调试测试文件

终端显示 Hello,World！说明你已经成功配置 C++环境了。
