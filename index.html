搭建个人博客的保姆级教程

（这是我折腾了很久，看了很多的教程，视频，碰了很多壁才总结出来的。结合各种教程来写出这篇我认为比较详细的方法。

​ 因为比较详细，所以文章比较长，请耐心浏览。）

这篇文章在2个教程基础上我做了部分修改以成功达成搭建个人博客网站的目的.教程1 教程2

我们将使用Hexo框架和GitHub来创建属于自己的博客。

一：搭建博客需要的工具：

​ 1：注册一个GitHub账号。注意！要记住你注册时填的邮箱和你的用户名！

​ GitHub账号注册详细流程

​ 2.下载Node.js和npm软件(npm包含其node.js里)。

​ Node.js官网

​ 3.下载Git软件（MacOS自带Git，不用下载）。

​ Git官网

Ⅰ：安装好Git和Node.js后打开CMD输入一下命令来检查是否安装成功。若出现类似v12.19.0这种版本号则说明安装成功。

​ 检查Node.js：node -v

​ 检查npm：npm -v

​ 检查Git：git –version

Ⅱ：安装淘宝镜像并下载Hexo。（以下3段代码均在终端界面输入。）

​ （用淘宝镜像是为了能够成功下载并安装所有重要程序。）

​ 1.安装淘宝镜像：npm install -g cnpm –registry=https://registry.npm.taobao.org

​ 2.安装Hexo：cnpm install -g hexo-cli

​ 3.测试：hexo -v 该代码输出结果大致如下图所示。

二： 搭建GitHub仓库。

​ 假设你已经注册好GitHub账号，现在创建仓库。步骤如下：

​ 1.按照图片进入仓库。进去后点写着New的绿色按钮。

​ 2.按了按钮后会进入到下图所示界面，按照图里设置。名字那里一定要输入你的GitHub名字.github.io然后点击Create repository。

三：生成SSH Keys。

​ 1.创建一个新文件夹来放你的博客文件。

​ 2.在你的博客文件夹里右击鼠标选Git Bash Here。

​ 3.输入**ssh-keygen -t rsa -C “你注册GitHub时用的邮箱地址” **，然后慢慢地按4次回车。结果如图所示。

​ 4.在 Users/用户名/ 下找到 .ssh 文件夹进入,打开 id_rsa.pub 文件（可以使用记事本，Notepad++或VS Code等软件打开）,并全部复制里面内容。

5.打开GitHub,添加新的SSH Key。步骤如下：

6.在Git Bash中输入代码测试ssh是否绑定成功：ssh -T git@github.com

​ PS:如果一直显示ssh: connect to host github.com port 22: Connection refused。就点击查看 教程

四：本地生成博客内容

1.在你的博客文件夹里打开Git Bash并输入hexo init

​ PS:输入该命令后可能出现连接超时（time out）的问题，若出现，则再尝试一次。实在不行就墙

成功生成的界面大致如下图：（当有INFO Start blogging with Hexo!时标志成功生成。）

这时，你的博客文件夹里就会有很多文件。

2.输入hexo s代码，过会后会显示如下界面。在浏览器中打开http://localhost:4000/，打开后就是你的本地博客界面啦。***记得输入后不要关闭Bash***，不然就打不开网页了。如果你要关闭网页的话按Ctrl+C就好了。

3.修改配置文件（PS:记得打开文件拓展名显示）

​ ①到你的博客文件夹，用记事本或VS Code等软件打 开.config.yml文件。

​ ②拉到最底下，记得切换成英文输入法

​ ③把最底下的几行代码替换成下面给出来的代码：(里面yourname替换成你的GitHub名字，其他不用换.)

deploy:

type: git

repository: git@github.com:yourname/yourname.github.io.git

branch: main

​ ④保存并退出文件。

4.在博客文件夹打开Git Bash，输入下面代码安装hexo-deployer-git自动部署发布工具。前面安装的淘宝镜像在这里发挥了作用。

​ cnpm install hexo-deployer-git –save

安装好hexo-deployer-git自动部署发布工具大致如下图所示：

5.在Github Bash界面输入生成页面的代码：hexo g

6.在Github Bash界面输入上传页面到GitHub代码：hexo d (前面安装hexo-deployer-git自动部署发布工具就是让这串代码可用。)

7.①输入代码过了一会儿后，他会让你输入用户名和密码,先输入用户名。

②输入用户名后先别急着输入密码，打开GitHub，生成一个新的token。步骤如下图：

③将复制的token粘贴到密码中。（建议记录生成的token，因为token只能显示一次。）

至此，你已经成功创建你的博客网站啦，这时候你就能访问你的网站啦。网站地址是：https://username.github.io

（username换成你的Github名字）
